## Overview

GitHub Actionsのデモ用。

## Commands

| コマンド        | 説明                                                  |
| --------------- | ----------------------------------------------------- |
| `build`         | TypeScript をビルド                                   |
| `check`         | リリース前チェック（name, lint, build, test）         |
| `clean`         | `lib/` ディレクトリを削除                             |
| `format`        | コードと `package.json` を自動整形                    |
| `lint`          | TypeScript 型チェック + Prettier フォーマットチェック |
| `rebuild`       | クリーンビルド                                        |
| `test`          | テスト実行（Vitest）                                  |
| `test:watch`    | テストの監視モード実行                                |
| `test:coverage` | カバレッジ付きテスト実行                              |
